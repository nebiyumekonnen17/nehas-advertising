# DESIGN PHILOSOPHY

## Less But Better

Remove unnecessary complexity.

Every element must justify its existence.

If something can be removed without harming usability, remove it.

---

## Clarity Above Everything

Every page should answer:

Where am I?

What can I do?

What should I do next?

within 3 seconds.

---

## Progressive Disclosure

Show advanced functionality only when needed.

Never overwhelm users.

---

## Consistency Is Mandatory

Buttons behave consistently.

Forms behave consistently.

Navigation behaves consistently.

Status indicators behave consistently.

Patterns should never surprise users.

# IMPLEMENTATION RULES

When modifying a page:

1. Improve UX if possible.
2. Improve responsiveness if possible.
3. Improve accessibility if possible.
4. Remove design debt.
5. Reuse existing design system.
6. Never create duplicate patterns.
7. Never sacrifice UX for implementation convenience.
